# **App Name**: GrocerEase

## Core Features:

- Product Browsing and Search: Browse products by category and search for specific items with filtering.
- Shopping Cart Management: Add, update, and remove items from the shopping cart. Calculate total cost dynamically.
- Checkout and Order Placement: Secure checkout process using Firestore to manage user data and facilitate payment.
- User Authentication: User signup/login with different roles (Customer, Admin).
- Product Management (Admin): Admin panel to add, update, and delete products.
- Order Management (Admin & Delivery): Admin and delivery roles can view and update order statuses.
- AI-Powered Review Summary: Use generative AI tool to summarize user reviews for each product, helping users quickly understand product feedback.

## Style Guidelines:

- Primary color: Fresh avocado green (#A3B18A) to evoke a sense of natural, healthy produce.
- Background color: Light sage (#F1F3EF), a desaturated version of the primary color to give a calming, organic feel.
- Accent color: Muted gold (#CBB26A), a sophisticated complement to the green tones.
- Headline font: 'Belleza', a serif font with personality, fitting with themes of fashion, art, and design
- Body font: 'Alegreya', a serif font paired with Belleza for readability in product descriptions and details.
- Use clean, minimalist icons representing grocery items, categories, and actions.
- Clean, grid-based layout optimized for both desktop and mobile devices, ensuring easy navigation and product discovery.
- Subtle transitions and animations on button hovers, product card interactions, and cart updates to enhance user experience.
